## Troubleshooting Tips (Summary)

**If Gemini doesn't remember something:**
- Create a GEMINI.md file with that information
- Use `/memory add` to teach it temporarily

**If you lose a conversation:**
- Use `/chat list` to find saved conversations
- If not saved, it's gone—remember to `/chat save`!

**If you're confused about a command:**
- Type `/help` to see all available commands
- Ask Gemini: "How do I use [command]?"

**If something doesn't work:**
- Ask your AI: "I tried [what you did]. It didn't work. Help?"
- AI companions are great at troubleshooting!
